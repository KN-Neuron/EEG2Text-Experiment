Eventful screen
===============

This class is a wrapper for a :doc:`screen <screens>` with an :doc:`event manager <event_managers>`. It is used by :doc:`experiment runner <experiment_runner>` and returned by :doc:`screen sequencers <screen_sequencers>`. It should not be subclassed or used on its own.

It can also be passed a callback to be called when the screen is shown.
